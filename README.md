# Sentiment analysis & topic modelling on Ryanair Reviews

University Project about Sentiment Analysis with fasttext embedding and Topic Modelling with BERT Topic (ROBERTA)

This project focuses on conducting an in-depth analysis of customer reviews for the airline Ryanair. By employing Natural Language Processing (NLP) and advanced Topic Modeling techniques, the goal is to extract actionable insights into customer satisfaction, identify prevailing service issues, and understand passenger sentiment.
